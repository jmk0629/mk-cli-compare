## 2026-06-22 / 시드 — CLI 실행 셸 인젝션 위험
- **증상(예방)**: 비교 엔진이 사용자 프롬프트를 받아 외부 CLI 를 실행한다.
- **원인**: `bash -c "claude -p \"$prompt\""` 처럼 셸 문자열로 조립하면 프롬프트에 `"; rm -rf` 등이 섞일 수 있음.
- **수정**: `ProcessBuilder(listOf(bin, "-p", prompt))` — 프롬프트는 **argv 원소 하나**로 전달. 셸 미경유.
- **재발방지 규칙**: CLI 실행은 항상 argv 리스트. 셸 보간 금지. 실행 바이너리는 카탈로그 whitelist 만.
