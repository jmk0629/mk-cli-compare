# log-mistake

실수를 `.claude/memory/MISTAKES.md` 에 영속 기록한다(repo 커밋 → 공유).

## 입력

- `symptom` — 무엇이 잘못됐나(관측된 증상/에러)
- `cause` — 근본 원인
- `fix` — 어떻게 고쳤나
- `rule` — 재발방지 규칙(다음부터 무엇을 다르게)

## 동작

1. `MISTAKES.md` 끝에 항목 추가 (최신이 아래로):
   ```
   ## YYYY-MM-DD / <한 줄 제목>
   - **증상**: ...
   - **원인**: ...
   - **수정**: ...
   - **재발방지 규칙**: ...
   ```
2. 날짜는 오늘. 같은 류 실수가 이미 있으면 새로 추가하지 말고 기존 항목의 규칙을 보강.
3. 규칙이 일반적이면 루트 `CLAUDE.md` 절대 규칙으로 승격 제안.

## 출력

추가한 항목 + (있다면) CLAUDE.md 승격 제안.
