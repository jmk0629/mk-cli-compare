# 2026-06-22 — 온보딩(사용법 3단계) 빈 상태

## 목표
- 첫 방문 모바일 사용자가 가치를 즉시 이해하도록 사용법을 안내 → 사용률 향상.

## 만든 것
- `CompareView` 에 **아직 비교 전(`!result && !running`)** 일 때만 보이는 3단계 안내 카드:
  1. ✍️ 프롬프트 입력(+모델 선택, ⌘/Ctrl+Enter)
  2. ⚡ 3 CLI 동시 실행(실시간 채워짐)
  3. 🏆 블라인드 투표 → 승률 리더보드
- 비교를 실행하면 자동으로 사라지고 결과가 그 자리를 차지.

## 검증 (라이브)
- 홈에서 3단계 카드 + 하단 최근 비교 피드 렌더(모바일 스크린샷), `next build` 통과.
