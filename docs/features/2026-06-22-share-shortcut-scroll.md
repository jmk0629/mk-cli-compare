# 2026-06-22 — 공유 버튼 + 단축키 + 결과 스크롤

## 목표
- 비교 결과를 쉽게 공유해 모바일 유입 증대. 입력 후 바로 실행/결과 확인하는 손맛.

## 만든 것
- **ShareButton**: 모바일은 `navigator.share`(네이티브 공유 시트), 데스크톱은 링크 클립보드 복사("✓ 링크 복사됨").
  - 비교 상세(`/comparison/[id]`) 헤더, 메인 결과(정체 공개 후)에 배치.
- **단축키**: 프롬프트 입력창에서 `⌘/Ctrl + Enter` → 비교 실행(placeholder 에 안내).
- **결과 스크롤**: 비교 생성 시 결과 영역으로 부드럽게 스크롤(`scroll-mt-20`) — 모바일에서 입력폼 아래 결과로 자연 이동.
